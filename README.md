# Éclat Cacao — Premium Artisan Chocolate

A minimalist, high-end landing page for Éclat Cacao, a fictional artisan chocolate brand. This project focuses on luxury aesthetics, premium typography, and smooth, scroll-driven interactions.

## Key Features

- **Cinematic Hero**: A static, high-impact editorial background with a sophisticated dark-gold theme.
- **Scroll-Driven Animation**: An interactive 35-frame image sequence in the CTA section that reveals a chocolate bar unwrapping as the user scrolls.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports.
- **Modern Performance**: Uses `IntersectionObserver` for entrance reveals and pre-loads animation assets for instant switching.

##  Technical Details

- **Structure**: Semantic HTML5.
- **Styling**: Vanilla CSS using custom variables and an 8px grid system.
- **Logic**: Vanilla JavaScript (no external dependencies).
- **Assets**: 35-frame JPG sequence and a custom PNG hero background.

##  Project Structure

- `index.html`: Main application file.
- `hero-bg.png`: Static hero background image.
- `hero-frames/`: Image sequence for the scroll animation.
